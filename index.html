📄 Cybersecurity Incident Report – Network Traffic Analysis

Resumo do Incidente

Usuários relataram indisponibilidade do site www.yummyrecipesforme.com. A análise via tcpdump indicou falha no serviço DNS.

Análise Técnica (Terminal Real)

$ tcpdump -i eth0 port 53

13:24:32.192571 IP 192.51.100.15.52444 > 203.0.113.2.domain: A? yummyrecipesforme.com
13:24:36.098564 IP 203.0.113.2 > 192.51.100.15: ICMP udp port 53 unreachable length 254

Protocolo: UDP
Serviço Afetado: DNS
Porta: 53
Erro: ICMP Port Unreachable

Conclusão Técnica

O servidor DNS não estava aceitando conexões na porta 53. Sem resolução DNS, o navegador não consegue localizar o servidor web. Não foram identificados indícios de ataque malicioso.
